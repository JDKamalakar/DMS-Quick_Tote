A high-productivity file access hub for **Dank Material Shell** (DMS), inspired by the ChromeOS Tote. Keep your most important files pinned and access recent downloads and screenshots instantly with native Material 3 aesthetics.

> [!TIP]
> You can drag and drop files directly from the Quick Tote into any application (like Discord or your Browser) for instantaneous sharing!

---

## ✨ Features

* **📌 Pinned Files:** Manually pin your most frequently used documents or folders for permanent one-click access.
* **📥 Recent Downloads:** Automatically monitors your downloads folder to display the latest files you've grabbed.
* **📸 Screen Captures:** Instant preview and access to your most recent screenshots.
* **🎨 Material Design:** Fluid animations, glassmorphism effects, and a responsive layout that fits perfectly into the DMS ecosystem.
* **⚙️ Configurable Limits:** Customize exactly how many items you want to keep visible in each section via the settings panel.

---

## 📸 Interface Sections

The tote organizes your workflow into logical groups for maximum efficiency:

| Section | Description |
| :--- | :--- |
| **Pinned** | Your manually curated workspace. Files here stay until you unpin them. |
| **Downloads** | A live feed of your `~/Downloads` (or custom path) folder. |
| **Screenshots** | High-visibility previews of your latest captures for quick sharing. |

## 🛠️ Installation

### 🚀 Recommended: DMS Plugin Manager
The easiest way to install and stay updated:
1. Open your **DMS Settings**.
2. Navigate to the **Plugin Manager** tab.
3. Search for `Quick Tote` and click **Install**.

### 🛠️ Manual Installation
For developers or users who want the latest edge builds:
1. Clone this repository into your DMS extensions/plugins folder:
   ```bash
   git clone https://github.com/JDKamalakar/DMS-Quick_Tote.git
   ```
2. Restart the DMS shell to enable the plugin.
